# Comece aqui

Este diretório é a memória oficial do Bona Prosa para desenvolvimento humano e com IA.

## Ordem de precedência

1. solicitação atual explícita dos fundadores;
2. `AGENTS.md`;
3. decisões aprovadas em `DECISIONS.md`;
4. documento específico do domínio;
5. código e testes existentes.

Se duas fontes oficiais divergirem, não adivinhe. Registre o conflito e peça confirmação.

## Mapa dos documentos

| Arquivo | Conteúdo |
| --- | --- |
| `PRODUCT.md` | Produto, público, MVP e limites |
| `PRICING.md` | Planos, preços e regras de cobrança |
| `BRAND.md` | Identidade visual, linguagem e acessibilidade |

| `DEVELOPMENT_WORKFLOW.md` | Como implementar e verificar tarefas |
| `DECISIONS.md` | Decisões aprovadas e substituições |
| `CHANGELOG.md` | Mudanças entregues no produto e nos documentos |

## Fluxo prioritário

`Landing → Cadastro → Verificação → Plano → Pagamento → Onboarding → Dashboard → Aula → Exercício → Feedback → Progresso`

Não expandir para social, empresas, vídeo ou personalização avançada antes de validar o fluxo prioritário.
